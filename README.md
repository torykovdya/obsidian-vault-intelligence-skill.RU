# 🧠 Obsidian Vault Intelligence — Claude Skill

> *Превращает ваши заметки в интеллектуальный капитал. Построено на методе Zettelkasten Никласа Луманна, настроено для AI-энтузиастов, создателей и автоматизаторов.*

---

## Что это такое

**Obsidian Vault Intelligence** — скилл для Claude, который анализирует ваш Obsidian-vault и отвечает на вопросы, которые обычные AI-инструменты игнорируют:

| Вопрос | Что делает скилл |
|--------|-----------------|
| В чём я реально становлюсь экспертом? | Строит карту компетенций по вашим заметкам |
| Какие темы я недоизучил? | Находит слепые зоны и пробелы |
| Какие статьи мне писать? | Генерирует идеи на основе ваших кластеров знаний |
| Какой продукт запускать? | Находит уникальные пересечения ваших навыков |
| Какие автоматизации создавать? | Выявляет паттерны "мануальной боли" |
| Кем я становлюсь через 6 месяцев? | Строит траекторию роста |

---

## Два режима работы

```
┌─────────────────────────────────────────────────────┐
│  LEVEL 1: Note Intelligence                          │
│  Вход: 1–10 заметок                                  │
│  Результат: Аудит, переписанные Permanent Notes,     │
│  связи, теги, маршрутизация в vault                  │
├─────────────────────────────────────────────────────┤
│  LEVEL 2: Vault Intelligence                         │
│  Вход: весь vault или большой дамп                   │
│  Результат: Карта знаний, карта компетенций,         │
│  возможности (статьи/продукты/автоматизации),        │
│  прогноз на 6 месяцев, health check                  │
└─────────────────────────────────────────────────────┘
```

## Быстрый старт

### 1. Установка скилла

Скачайте файл `obsidian-vault-intelligence.skill` из [Releases](../../releases) и установите в Claude.

### 2. Первый анализ (5 минут)

Скопируйте 5–10 своих заметок из Obsidian и отправьте Claude:

```
Вот мои заметки из Obsidian. Проанализируй их и скажи:
1. Какой тип каждая заметка?
2. Как их улучшить?
3. Что они говорят о моих интересах?

[вставьте заметки]
```

### 3. Полный анализ vault (20–30 минут)

Экспортируйте vault в текст и отправьте:

```
Вот мой полный Obsidian vault (или его часть).
Сделай Vault Intelligence анализ: карту знаний, компетенций,
возможности и прогноз на 6 месяцев.

[вставьте содержимое vault]
```

## Философия

Этот скилл построен на трёх принципах из книги Зонке Аренса *"How To Take Smart Notes"*:

**1. Заметки ценны только в сети связей**
> Одиночная заметка — это факт. Связанная заметка — это знание. Тысяча связанных заметок — это интеллектуальный капитал.

**2. Инсайты нельзя запланировать, но можно создать для них условия**
> Zettelkasten — не архив. Это среда, в которой идеи сталкиваются и порождают новые идеи.

**3. Ваши заметки предсказывают, что вы можете создать**
> Луман написал 58 книг, делая 6 заметок в день. Не потому что был гением — потому что его система показывала ему, что он уже знает.
