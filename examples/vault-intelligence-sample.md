### 🚀 Продукты / Проекты

```
ТОП-3 ПРОДУКТОВЫХ ВОЗМОЖНОСТИ:

1. "Prompt Engineering Audit" — productized консалтинг
   Тип: Сервис (2–3 часа работы)
   Основан на: ваша глубина в промптинге уникальна

   Тип: Digital product (набор шаблонов + гайд)
   Основан на: 11 заметок по n8n = готовые паттерны
   MVP за 2 недели: 5 готовых n8n workflows + PDF-гайд
   Цена: $29–49
   Кому: non-technical founders, которые хотят автоматизировать работу с AI

3. "Weekly AI Digest" — кураторский newsletter
   Тип: Newsletter (монетизация через спонсоров или платный уровень)
   Основан на: ваша система потребления + структурирования информации
   MVP за 2 недели: первые 3 выпуска, запуск на Substack
   Модель: бесплатно → платный уровень с разборами
```

### ⚡ Автоматизации для создания

```
ТОП-3 АВТОМАТИЗАЦИИ (на основе паттернов в заметках):

1. "AI Content Pipeline" в n8n
   Решает: вы упоминаете "трачу много времени на поиск идей"
   Логика: RSS лучших AI-источников → Claude-суммаризация → Obsidian Inbox
   Стек: n8n + Claude API + Obsidian (через локальную папку)
   Сложность: средняя (2–3 дня)

2. "Fleeting Notes Processor"
   Решает: Inbox накапливается, обрабатывать лень
   Логика: голосовая заметка → транскрипция → Claude классификация → Obsidian
   Стек: Whisper API + Claude + n8n + Obsidian
   Сложность: средняя

3. "Content Repurposing Agent"
   Решает: одна статья → пропадает, хотя можно переупаковать
   Логика: статья → Claude → 5 вариантов постов для разных платформ
   Стек: Claude API + n8n + Buffer/Typefully
   Сложность: низкая (1 день)
```
