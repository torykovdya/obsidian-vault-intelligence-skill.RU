# AI Creator Taxonomy

Use this reference when building competency maps and opportunity analysis for AI enthusiasts.

---

## Core Skill Domains for AI Builders

### Technical Foundation
- LLM Architecture (transformers, context, tokenization, fine-tuning)
- Prompt Engineering (few-shot, CoT, structured output, system prompts)
- AI Agents (ReAct, tool use, memory, orchestration frameworks)
- RAG & Knowledge Systems (embeddings, vector DBs, chunking strategies)
- Automation & Workflows (n8n, Make, Zapier, custom APIs)
- Evals & Testing (LLM-as-judge, benchmarks, red-teaming)
- Multimodal (vision, audio, video generation)
- Local AI (Ollama, fine-tuning on own hardware)

### Creator/Business Skills
- Content Strategy (what to create, for whom, in which format)
- Distribution (SEO, social, newsletter, community)
- Audience Building (trust loops, positioning, consistency)
- Product Thinking (problem → solution → go-to-market)
- Pricing & Monetization (SaaS, info products, consulting, agencies)
- No-Code / Low-Code Building (rapid prototyping)

### Meta-Skills
- Learning System Design (how you capture and connect knowledge)
- Research Methods (how you validate ideas)
- Network Effects (who to know, how to build credibility)

## AI Creator Archetypes

Map the user's notes to the archetype they're heading toward:

| Archetype | Core skills | Primary output | Monetization |
|-----------|------------|----------------|--------------|
| **AI Educator** | Prompting, explaining, curriculum | Courses, tutorials, YouTube | Cohorts, courses, sponsorships |
| **AI Tool Builder** | Agents, APIs, no-code | SaaS tools, browser extensions | Subscriptions, one-time |
| **AI Automator** | Workflows, integrations | Automation services, templates | Agency, productized service |
| **AI Researcher/Writer** | Deep analysis, synthesis | Newsletter, essays, reports | Newsletter paid, consulting |
| **AI Community Builder** | Curation, networking | Slack/Discord, events | Membership, sponsorships |
| **AI Consultant** | Strategy, implementation | Client projects | Retainers, project fees |
| **Niche AI Specialist** | Domain (legal/health/finance) + AI | Vertical tools | B2B SaaS, consulting |

## Rarity Matrix (for opportunity detection)

Skills that are **rare in combination** are most valuable:

```
Technical Depth × Niche Domain        = High value B2B opportunity
Technical Depth × Content Skill       = Thought leadership
Automation Know-how × Specific Stack  = Productized service
Deep Research × Distribution          = Newsletter / media business
LLM Fine-tuning × Specific Industry   = Vertical AI product
```

## Content Types for AI Creators

When suggesting articles, pick the format that matches note density:

| Note density | Best format |
|-------------|-------------|
| 1–3 notes on topic | Social post / thread |
| 4–8 notes | Blog post / essay |
| 9–15 notes | Deep dive / report |
| 15+ notes | Course module / book chapter |
| Cross-cluster insight | Signature piece / framework |

## Automation Opportunity Signals in Notes

Look for these patterns → suggest automation:

- "Every week I manually..." → workflow automation
- "I wish there was a tool that..." → build it
- "I keep doing the same thing in [tool]..." → template or macro
- "It takes me X hours to research..." → AI-assisted research pipeline
- "I aggregate data from..." → automated dashboard
- "I get asked this question all the time..." → chatbot / FAQ agent
